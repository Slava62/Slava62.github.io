Программа DrillSplitter

В процессе сверления большого количества отверстий на станках с ЧПУ возникает задача останова выполнения управляющей программы станка и выхода в определенную позицию для контроля состояния сверла. Современные CAM-системы не всегда позволяют вставить блоки кода для разбиения управляющей программы (УП) станка, поэтому их приходится вставлять вручную. В том случае, когда УП рассчитана на 1000 — 2000 отверстий, и выход нужно осуществлять каждые 50 отверстий, а также таких УП много, возникает задача автоматизации вставки блоков. Этот процесс реализован в данной программе.

В качестве исходных данных приложение использует текстовый файл УП. По заданному алгоритму происходит поиск мест вставки блоков. После этого УП можно сохранить. Также есть возможность изменить параметры блока.
